Información del elemento datalist

Este elemento especifica una lista de opciones predefinidas para un elemento input, ofrece una función de autocompletado para este. Los usuarios verán una lista desplegable de las opciones a medida que van insertando datos.

La id del elemento datalist debe ser igual al id del elemento input.

Información del atributo multiple

Este atributo de tipo booleano le permite al usuario añadir o seleccionar más de un valor. Puede ser usado con los elementos input y select. Para seleccionar varios elementos tenemos que mantener pulsada la tecla Control.

Información del atributo autofocus

Este atributo de tipo booleano especifica que el elemento donde este situado será seleccionado cuando la página cargue. El atributo autofocus puede ser usasdo en los siguientes elementos:

  • button
  • input
  • select
  • textarea
